# 等一個人的咖啡

等一個人的咖啡是一個用 Email 推動流程、以柔焦照片與三個問題降低外貌篩選，並協助雙方在安全公開地點完成第一次約 1 小時咖啡快聊的交友網頁服務。

目前專案先從產品規劃開始：

- [GitHub Pages 首頁原型](index.html)
- [正式 PRD、使用者流程、MVP scope、資料庫設計與開發順序](docs/coffee-meet-prd.md)
- [MVP 靜態原型](prototype/index.html)

若已啟用 GitHub Pages，可從這裡觀看：

https://jbuduoo.github.io/CoffeeMeet/
